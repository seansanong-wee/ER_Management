วิธีเชื่อมต่อข้อมูลจาก Google Sheet

ระบบจะพยายามดึงข้อมูลจาก Google Apps Script ตาม URL ที่ตั้งไว้ หากดึงไม่ได้ ระบบจะแสดงข้อมูลตัวอย่างเพื่อให้หน้าจอยังใช้งานได้

ขั้นตอนที่ 1: สร้าง API ด้วย Google Apps Script

  1. เปิด Google Sheet แล้วไปที่เมนู ส่วนขยาย > Apps Script
  2. วางโค้ดฟังก์ชัน doGet(e) เพื่อส่งข้อมูลเป็น JSON
  3. กด Deploy > New deployment > เลือก Web app
  4. ตั้งค่า Who has access เป็น Anyone
  5. คัดลอก Web app URL มาใส่ในตัวแปร GAS_API_URL

ขั้นตอนที่ 2: ตั้งค่าในไฟล์นี้

ค้นหาส่วน JavaScript ด้านล่าง แล้วกำหนดค่าให้ตรงกับ URL ของคุณ

const USE_GAS_API = true;
const GAS_API_URL = 'วางลิงก์ Web App URL ตรงนี้';
